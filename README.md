## 📋 Системные требования

| Компонент | Минимальные требования | Рекомендуемые требования |
|-----------|------------------------|--------------------------|
| **ОС** | Windows 10 (22H2) | Windows 11 |
| **Процессор** | * | * |
| **Оперативная память** | 4 ГБ | 8 ГБ |
| **Видеокарта** | * | * |
| **DirectX** | Версия 12 | Версия 12 |
| **Сеть** | Широкополосное подключение к интернету | Широкополосное подключение к интернету |

## 🔧 Установка и запуск

1. Распакуйте архив с билдом в удобное место
2. Для запуска клиента используйте `WindowsClient/client.bat`
3. Для запуска сервера используйте `WindowsServer/server.bat`

> **Примечание:** Убедитесь, что порт 7777 открыт в брандмауэре для корректной работы сетевой игры.
